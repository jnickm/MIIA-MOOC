Utilizando nuestro mouse podríamos movernos por la plataforma de Jupyter de una forma básica. Sin embargo, les aseguramos que, al menos por la forma en que se desarrolló el curso, van a pasar mucho tiempo utilizando la plataforma, por lo tanto, la velocidad que ofrece el mouse se va a quedar corta y van a querer moverse con mucha más fluidez sobre Jupyter. Entonces, existen dos modos de interacción con el teclado cuando estamos trabajando en Jupyter. Cada una de estos cuentan con sus propios atajos y funciones. Dichas modos son: Modo edición y Modo comando.
Si se desea ver la totalidad de atajos, se puede visualizar uno para mostrar la lista de atajos para tu sistema operativo, presionando H. Pero si este no funciona puedes acceder a la lista dando click donde se muestra en la siguiente imagen. También puedes personalizarlos si deseas.

1. Modo comando.

Este modo se activa presionando la tecla Esc y/o dando click sobre la celda la cual se quiere interactuar. Sabremos que estamos en el modo comando cuando la izquierda de la celda tenga una barra de color azul.
Se puede subir una imagen de la barra de color azul o inserta una celda de code como ejemplo

Atajos para correr código o representar texto.

Empecemos esta sección con los que probablemente serán (o son) su mejor opción para evaluar el contenido que se ha escrito en la celda:
  • Control + Enter : evalúa el contenido de la celda y permanece en ella.
  • Shift + Enter : evalúa el contenido de la celda y se mueve hacia la siguiente, creando una nueva si no hay otra celda a la cual se pueda mover.

Movimiento sobre el Notebook.
  • ↓↓: Se desplaza hacia la celda inferior.
  • ↑↑: Se desplaza hacia la celda superior.
  • A: Añadir celda sobre la celda actual.
  • B: Añadir celda bajo la celda actual.
  • D + D: Elimina la celda actual.
  • Z : Deshace la eliminación de una celda.
  • M : Convierte la celda actual en una celda Markdown.
  • C : Convierte la celda actual en una celda de codigo.

2. Modo edición.

Este modo es el que se activa cuando estamos creando contenido en una casilla, que por cierto se crea haciendo doble click sobre la celda que se desee editar. Sabemos que está activo, pues además de darnos la opción de escribir, la barra del lado izquierdo tendrá un color verde. Para este tipo de atajos es recomendable ver la lista que ofrece Jupyter debido a que estos pueden cambiar mucho entre sistemas operativos. Aunque de igual forma aclaramos que en el modo de edición generalmente puedes utilizar los atajos que uses en cualquier editor de texto, algunos de estos son:
  • Ctrl+C: Copiar
  • Ctrl+V: Pegar
  • Ctrl+X: Cortar
  • Ctrl+Z: Deshacer

1. Celdas Markdown.

Este tipo de celdas son las encargadas de almacenar el texto del en el documento, crear tablas, insertar imágenes, entre otras funciones. Es generalmente utilizado para la descripción, o explicación del código o resultados que se presentan en su(s) casilla(s) inferior(es).
Este tipo de celdas requieren de diferentes recursos para dar formato al texto que se va a presentar. Estos recursos son interpretados por el notebook para dar un resultado final, como puede ser agregar texto en negrilla, creación de formulas matemáticas: ∑𝑖=1𝑁𝜅=𝑛(𝑛+1)2∑i=1Nκ=n(n+1)2, o definición de títulos, sub-títulos, entre otras funciones que se explicarán a continuación.
